# Formare

## Description
- Formare is a Catholic-inspired menstrual cycle tracking application built with a Node.js full-stack architecture. It helps users track primary and secondary biomarkers associated with the menstrual cycle while encouraging awareness of the body through a lens consistent with Natural Law and Natural Family Planning (NFP) principles.

Formare (Latin): “to form,” “to shape,” “to develop”

This app is designed primarily for cycle awareness and health tracking, not pregnancy avoidance or contraception planning.

## Tech Stack
- React
- Node.js
- MongoDB
- PM2
- Nginx
- GCP

## Features
- Cycle tracking
- User authentication
- Data visualization
