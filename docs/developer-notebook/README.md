# 🌸 Formare

**Formare** is a Catholic-inspired menstrual cycle tracking application built with a **Node.js full-stack architecture**. It helps users track primary and secondary biomarkers associated with the menstrual cycle while encouraging awareness of the body through a lens consistent with **Natural Law** and **Natural Family Planning (NFP) principles**.

> *Formare* (Latin): “to form,” “to shape,” “to develop”

This app is designed primarily for **cycle awareness and health tracking**, not pregnancy avoidance or contraception planning.

## ✨ Features

### 📊 Primary Biomarker Tracking
Users can log:

- 🌡️ Basal Body Temperature (BBT)
- 💧 Cervical Mucus (CM)
- 🩸 Bleeding patterns

These help provide insight into cycle phases and patterns over time.

### 🧩 Secondary Biomarker Tracking

Optional symptom tracking includes:

- 🤕 Headaches
- 🪑 Back aches
- 😴 Fatigue
- 😌 Mood changes
- 📌 Custom symptoms (planned feature)

---

### 📅 Cycle Awareness Tools

Formare provides:

- Estimated fertile window indicators
- Estimated cycle phase tracking
- Period predictions based on historical data
- Long-term cycle pattern visibility

⚠️ Predictions are informational only and **not intended for pregnancy planning decisions**

## ✝️ Mission & Philosophy

Formare exists to support a vision of health tracking rooted in:

- dignity of the human person
- respect for the natural rhythms of the body
- non-contraceptive cycle awareness
- integration of faith and technology

It is inspired by principles found in **Natural Family Planning**, but it is **not an NFP decision-making tool**.
